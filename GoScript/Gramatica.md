# Gramática Libre de Contexto - GoScript

A continuación se presenta el detalle de las producciones gramaticales utilizadas en el analizador sintáctico (Jison) para el lenguaje GoScript.

---

## 1. Estructura Principal

```ebnf
INICIO : INSTRUCCIONES EOF 

INSTRUCCIONES : INSTRUCCIONES INSTRUCCION    
              | INSTRUCCIONES PUNTO_COMA     
              | INSTRUCCION                  
              | PUNTO_COMA                   
```

## 2. Instrucciones (Sentencias)

```ebnf
INSTRUCCION : INS_IMPRIMIR 
            | ASIGNAR    
            | DECLARACION    
            | INS_IF 
            | ASIGNACION_COMPUESTA 
            | DEC_CORTA 
            | INS_FOR 
            | INS_BREAK 
            | INS_CONTINUE 
            | INS_SWITCH 
            | INS_FUNCION 
            | INS_RETURN 
            | INS_LLAMADA_INST 
            | INS_INCREMENTO 
            | INS_DECREMENTO 
            | STRUCT_DEF 
            | LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 
```

## 3. Declaraciones y Asignaciones

```ebnf
DECLARACION : VAR ID TIPO 
            | VAR ID TIPO IGUAL EXPRESION 
            | TIPO_BASE ID IGUAL EXPRESION 
            | TIPO_BASE ID 
            | ID ID IGUAL EXPRESION 
            | ID ID 
            | CORCHETE_ABRE CORCHETE_CIERRA TIPO ID IGUAL EXPRESION
            | CORCHETE_ABRE CORCHETE_CIERRA TIPO ID

DEC_CORTA : ID DOS_PUNTOS_IGUAL EXPRESION

ASIGNAR : EXPRESION IGUAL EXPRESION 

ASIGNACION_COMPUESTA : ID MAS_IGUAL EXPRESION 
                     | ID MENOS_IGUAL EXPRESION 

INS_INCREMENTO : ID MAS_MAS 

INS_DECREMENTO : ID MENOS_MENOS 
```

## 4. Estructuras de Control de Flujo

### Condicionales (If - Switch)
```ebnf
INS_IF : IF EXPRESION LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 
       | IF EXPRESION LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA ELSE LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 
       | IF EXPRESION LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA ELSE INS_IF 

INS_SWITCH : SWITCH EXPRESION LLAVE_ABRE CASOS LLAVE_CIERRA 
           | SWITCH EXPRESION LLAVE_ABRE CASOS DEF_BLOCK LLAVE_CIERRA 
           | SWITCH EXPRESION LLAVE_ABRE DEF_BLOCK LLAVE_CIERRA 

CASOS : CASOS CASO 
      | CASO       

CASO : CASE EXPRESION DOS_PUNTOS INSTRUCCIONES 
     | CASE EXPRESION DOS_PUNTOS 

DEF_BLOCK : DEFAULT DOS_PUNTOS INSTRUCCIONES 
          | DEFAULT DOS_PUNTOS 
```

### Ciclos y Transferencia (For, Break, Continue, Return)
```ebnf
INS_FOR : FOR EXPRESION LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 
        | FOR INSTRUCCION PUNTO_COMA EXPRESION PUNTO_COMA INSTRUCCION LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 
        | FOR ID COMA ID DOS_PUNTOS_IGUAL RANGE EXPRESION LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 

INS_BREAK : BREAK 

INS_CONTINUE : CONTINUE 

INS_RETURN : RETURN EXPRESION 
           | RETURN 
```

## 5. Funciones y Llamadas

```ebnf
INS_FUNCION : FUNC ID PAR_ABRE PARAMETROS PAR_CIERRA TIPO_RETORNO LLAVE_ABRE INSTRUCCIONES LLAVE_CIERRA 

PARAMETROS : PARAMETROS COMA PARAMETRO 
           | PARAMETRO 
           | EPSILON 

PARAMETRO : ID TIPO 

TIPO_RETORNO : TIPO 
             | EPSILON

INS_LLAMADA_INST : LLAMADA_EXPR 

LLAMADA_EXPR : ID PAR_ABRE ARGUMENTOS PAR_CIERRA 

ARGUMENTOS : ARGUMENTOS_LISTA 
           | ARGUMENTOS_LISTA COMA 
           | EPSILON

ARGUMENTOS_LISTA : ARGUMENTOS_LISTA COMA EXPRESION 
                 | EXPRESION 
```

## 6. Structs (Estructuras)

```ebnf
STRUCT_DEF : STRUCT ID LLAVE_ABRE ATRIBUTOS_DEF LLAVE_CIERRA 

ATRIBUTOS_DEF : ATRIBUTO_DEF 
              | ATRIBUTOS_DEF ATRIBUTO_DEF 

ATRIBUTO_DEF : TIPO ID PUNTO_COMA 

VALORES_STRUCT : VALORES_STRUCT_LIST 
               | VALORES_STRUCT_LIST COMA 
               | EPSILON

VALORES_STRUCT_LIST : VALOR_STRUCT 
                    | VALORES_STRUCT_LIST COMA VALOR_STRUCT 

VALOR_STRUCT : ID DOS_PUNTOS EXPRESION 
```

## 7. Expresiones y Operaciones

```ebnf
EXPRESION : EXPRESION MAS EXPRESION  
          | EXPRESION MENOS EXPRESION  
          | EXPRESION POR EXPRESION  
          | EXPRESION DIVIDIDO EXPRESION  
          | EXPRESION MODULO EXPRESION  
          | MENOS EXPRESION %prec UMENOS 
          | ENTERO             
          | DECIMAL          
          | CADENA            
          | L_RUNE            
          | NIL               
          | RTRUE            
          | RFALSE            
          | EXPRESION IGUAL_IGUAL EXPRESION 
          | EXPRESION DIFERENTE EXPRESION 
          | EXPRESION MENOR EXPRESION 
          | EXPRESION MENOR_IGUAL EXPRESION 
          | EXPRESION MAYOR EXPRESION 
          | EXPRESION MAYOR_IGUAL EXPRESION 
          | EXPRESION AND EXPRESION 
          | EXPRESION OR EXPRESION 
          | NOT EXPRESION 
          | LLAMADA_EXPR 
          | ID         
          | PAR_ABRE EXPRESION PAR_CIERRA 
          | CORCHETE_ABRE CORCHETE_CIERRA TIPO LLAVE_ABRE ARGUMENTOS LLAVE_CIERRA
          | LLAVE_ABRE VALORES_STRUCT LLAVE_CIERRA 
          | LLAVE_ABRE ARGUMENTOS LLAVE_CIERRA 
          | EXPRESION CORCHETE_ABRE EXPRESION CORCHETE_CIERRA 
          | EXPRESION PUNTO ATTR_ID 
          | APPEND PAR_ABRE EXPRESION COMA ARGUMENTOS PAR_CIERRA 
          | STRCONV_ATOI PAR_ABRE EXPRESION PAR_CIERRA 
          | STRCONV_PARSEFLOAT PAR_ABRE EXPRESION PAR_CIERRA
          | REFLECT PUNTO TYPEOF PAR_ABRE EXPRESION PAR_CIERRA 
          | LEN PAR_ABRE EXPRESION PAR_CIERRA 
          | SLICE_INDEX PAR_ABRE EXPRESION COMA EXPRESION PAR_CIERRA 
          | STRINGS_JOIN PAR_ABRE EXPRESION COMA EXPRESION PAR_CIERRA 

EXPRESION_ACCESO_SLICE : EXPRESION CORCHETE_ABRE EXPRESION CORCHETE_CIERRA 
```

```ebnf

```
